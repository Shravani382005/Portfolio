# **App Name**: Shravani's Digital Canvas

## Core Features:

- Dynamic Project Showcase: Display projects with images, titles, descriptions, tech stacks, and links.
- Interactive Skills Display: Showcase skills and technologies with icons and labels.
- Contact Form Integration: Enable visitors to send direct messages via an integrated contact form.
- Personalized Intro Generation: AI tool generates a unique and engaging introductory blurb for the hero section, highlighting key skills and experience. The AI will be used as a tool to come up with fresh descriptions from time to time.
- Live Project Data Updates: Automatically update project information on the website when changes are made in the data file.
- Responsive Layout: Ensure the website adapts to different screen sizes, providing a consistent user experience across devices.
- External Links: Allow user to reach out via email or Linkedin

## Style Guidelines:

- Primary color: Soft violet (#B28FCE) to establish a calm and modern aesthetic, inspired by lavender fields.
- Background color: Light, desaturated violet (#F0EDF5) to provide a clean, airy feel.
- Accent color: A slightly brighter, analogous purple (#9A79BA) will serve as a visual contrast for primary actions and highlights
- Headline font: 'Space Grotesk' (sans-serif) for modern and tech-forward titles; body font: 'Inter' (sans-serif) for clean and readable content.
- Utilize Font Awesome or Bootstrap Icons for a consistent and modern visual language.
- Employ clean sections, modern spacing, and rounded cards to mirror the Dribbble design.
- Add smooth hover animations and subtle reveal animations on scroll to enhance user engagement.
